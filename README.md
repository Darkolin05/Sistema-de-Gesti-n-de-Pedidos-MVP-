# Sistema Integrado de Gestión de Pedidos y Control de Inventario

## Descripción
Este proyecto es una aplicación web desarrollada para optimizar la toma de pedidos y el control de inventarios en restaurantes, reemplazando procesos manuales por una solución tecnológica eficiente.

## Historias de Usuario
1. **HU-01:** Como mesero, quiero registrar un nuevo pedido para comenzar a atender al cliente.
2. **HU-02:** Como mesero, quiero agregar múltiples platos al pedido para completar la orden del cliente.
3. **HU-03:** Como cocinero, quiero visualizar los pedidos pendientes para saber qué debo preparar.
4. **HU-04:** Como cocinero, quiero marcar un pedido como "completado" para notificar al mesero.
5. **HU-05:** Como administrador, quiero visualizar el resumen de ventas para tomar decisiones estratégicas.

## Tecnologías Utilizadas
- **React.js** (vía CDN)
- **Tailwind CSS** (vía CDN)
- **JavaScript (ES6+)**
- **Git/GitHub** para control de versiones

## Instrucciones de Instalación
1. Clona este repositorio: `git clone <url-de-tu-repo>`
2. Abre el archivo `index.html` en tu navegador favorito.
3. No requiere instalación de servidores adicionales al ser una aplicación cliente.

## Mantenimiento
Este proyecto utiliza ramas (`branches`) para cada funcionalidad, garantizando que el código base en `main` siempre esté estable.
